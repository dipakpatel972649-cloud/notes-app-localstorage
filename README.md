📝 Notes App

A simple and responsive Notes Application built using HTML, CSS, JavaScript, and Bootstrap.
This app allows users to create, edit, delete, and manage notes directly in the browser using localStorage.

✨ Features:-

➕ Add new notes

✏️ Edit existing notes

🗑️ Delete notes

🎨 Random colored note cards

💾 Notes stored in browser localStorage

📱 Responsive design

📂 Folder style UI layout

🔍 Search input UI
